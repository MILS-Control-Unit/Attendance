Upload an Excel sheet (.xlsx/.xls) with columns: Name, Section, Stage, Grade, Subject, Classes, Note. Stage must be one of KG / Primary / Prep. / Secondary, and Grade must match a valid grade for that stage. Imported teachers are added to the database and instantly linked to the Absence and Summary sheets, same as manual entries. Names already stored in the database (or repeated inside the file) are automatically skipped and not re-added.
